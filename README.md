# 🌴 Echoes of Kerala

A visually rich cultural heritage website built using **HTML5** and **CSS3**, showcasing the beauty, history, traditions, cuisine, and architecture of Kerala.

## 📖 Overview

**Echoes of Kerala** is a static web project designed to celebrate Kerala's cultural identity through an elegant and responsive user interface. The website provides visitors with an immersive journey through Kerala's hidden destinations, traditional art forms, historical timeline, cuisine, and architectural heritage.

## ✨ Features

* 🎨 Modern and responsive UI design
* 🏞️ Showcase of Kerala's hidden gems
* 📜 Interactive heritage timeline
* 💃 Traditional art forms section
* 🍛 Kerala cuisine highlights
* 🏛️ Heritage architecture gallery
* 📱 Mobile-friendly design
* ⚡ Smooth scrolling navigation
* 🌄 High-quality visual content

## 🛠️ Technologies Used

* Responsive Design Principles

## 🎯 Sections Included

### Home

A full-screen hero section introducing Kerala's cultural richness.

### Hidden Gems

Highlights lesser-known destinations including:

* Meesapulimala
* Kumbalangi
* Bekal Fort

### Heritage Timeline

Presents important historical phases of Kerala:

* Ancient Kerala
* Spice Trade Era
* Colonial Period
* Modern Kerala

### Traditional Art Forms

Features:

* Kathakali
* Mohiniyattam
* Theyyam

### Kerala Cuisine

Showcases famous dishes:

* Sadya
* Appam & Stew
* Puttu & Kadala

### Kerala Heritage Gallery

A visual collection representing Kerala's traditional architecture and cultural landmarks.
